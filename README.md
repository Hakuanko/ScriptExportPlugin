# ScriptExportPlugin for YMM4

ゆっくりMovieMaker4 (YMM4) のタイムライン上にあるボイスアイテムを、  
**YMM4 台本 TXT 形式**のテキストファイルとしてエクスポートするプラグインです。

AI（Claude）を使って作りました。

---

## 概要

YMM4 でボイスアイテムを手動で並べた後、その内容を台本ファイルとして書き出したいケースがあります。  
このプラグインは YMM4 の「ファイル」メニューに **「台本をテキストで出力」** 項目を追加し、  
1クリックで台本ファイルを生成できるようにします。

出力形式は YMM4 標準の台本 TXT 形式に準拠しているため、  
他のプロジェクトへの台本インポートや、台本の外部編集・管理にそのまま活用できます。

---

## 導入方法

### 動作環境

- ゆっくりMovieMaker4 最新版
- .NET 10 以上

### ビルド＆インストール手順

1. `ScriptExportPlugin.csproj` を開き、`YMM4DirPath` を自分の環境に合わせて書き換えます。
   ```xml
   <YMM4DirPath>C:\YukkuriMovieMaker_v4_Lite\</YMM4DirPath>
   ```
2. ビルドします（Visual Studio または dotnet CLI）。
   ```bash
   dotnet build -c Release
   ```
3. ビルド成功と同時に、`YMM4DirPath\user\plugin\ScriptExportPlugin\` へ DLL が自動コピーされます。
4. YMM4 を起動（または再起動）します。
5. ファイルメニューに **「台本をテキストで出力」** が表示されれば導入完了です。

---

## 使い方

1. YMM4 でプロジェクトを開き、タイムラインにボイスアイテムを配置します。
2. メニューバーの **「ファイル」→「台本をテキストで出力」** をクリックします。
3. 保存先ファイルを指定し、**「保存」** をクリックします。
4. 指定したパスに台本 TXT ファイルが生成されます。

## 機能説明

### 出力フォーマット

[YMM4 公式の台本 TXT 形式](https://manjubox.net/ymm4/faq/editing/台本ファイルをもとにボイスアイテムを追加する/) に準拠しています。

```
キャラクター名「セリフ内容」
```

**例：**

```
キャラ１「こんにちは！キャラ１なのだ。」
キャラ２「今日もよろしくお願いしますだわ。」
キャラ１「セリフ内で\「カギ括弧\」を使う場合はこうなるのだ。」
```

### 主な挙動

| 項目 | 内容 |
|------|------|
| 対象アイテム | VoiceItem のみ（テキスト・画像等は除外） |
| 出力順 | タイムライン上の開始フレーム昇順（時系列順） |
| キャラクター名取得 | `Character.Name` → `Description` の順でフォールバック |
| カギ括弧エスケープ | セリフ内の `「」` を自動的に `\「\」` に変換 |
| セリフ内改行 | そのまま保持（YMM4 仕様） |
| 文字コード | UTF-8 |

---

## 注意点

- **YMM4 の内部 API はリフレクションで取得しています。**  
  YMM4 のバージョンアップにより内部構造が変更された場合、正常に動作しなくなる可能性があります。  
  その際は `VoiceLineCollector.cs` 内のプロパティ名定数を更新してください。

- プロジェクトが開かれていない状態で実行すると、「ボイスアイテムが見つかりません」と表示されます。

- キャラクター名が取得できない場合は `不明` として出力されます。

- セリフが空白のみのボイスアイテムは出力対象外となります。

## プロジェクト構成

```
ScriptExportPlugin/
├── ScriptExportPlugin.cs   # IPlugin エントリポイント・イベントルーティング
├── MenuInjector.cs         # ファイルメニューへの項目挿入
├── VoiceLineCollector.cs   # タイムラインからのボイスライン収集・フォーマット
├── ScriptFileWriter.cs     # ファイル保存ダイアログ・書き出し
└── VisualTreeHelper.cs     # WPF ビジュアルツリー探索ユーティリティ
```
